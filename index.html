Welcome to Membles.

Usage: membles -t trace -d spec/device.spec [-s ctrl/system.ctrl] [-o output] [-v] [-h]

       -t, --trace=FILE                  specify a trace file to run
       -d, --device=FILE1[,FILE2,...]    specify a list of device configurations
       -c, --ctrl=FILE1,[,FILE2,...]     specify a list of controller configurations
       -o, --output=FILE                 specify a file name for all the outputs
                                         e.g. FILE.log, FILE.csv, and FILE.trc
       -v, --verbose                     enable verbosity
       -h, --help                        print this message
